# Notificaciones-red-social
Este proyecto tiene como objetivo desarrollar un sistema que envie notificaciones de las interacciones en una red social en tiempo real mediante un modelo de IA 

#Arquitectura
 La arquitectura del sistema es Lambda por su capacidad de manejar datos en tiempo real a baja latencia.

 #Entorno tecnico:
-Git 
-Python 
-Docker 
-Kafka 
-Druid
